# 블록AI (BlockAI)

수행평가시 AI·챗봇 사이트를 감지하고 자동으로 브라우저를 종료해 주는 가벼운 Tauri 기반 도구입니다.

## 주요 기능

-   **실시간 감시**: 활성 창 제목을 초 단위로 스캔하여 지정된 키워드가 포함된 브라우저를 감지합니다.
-   **동적 키워드 업데이트**: `https://github.com/banatic/blockAI/keyword.toml`에서 실시간으로 차단 목록을 동기화합니다.

## 시작하기

1.  본 저장소를 클론합니다.
2.  `npm install`으로 의존성 패키지를 설치합니다.
3.  `keyword.toml` 파일에 차단하고자 하는 키워드를 추가합니다.
4.  `npm run tauri dev`를 통해 실행하거나 빌드합니다.
